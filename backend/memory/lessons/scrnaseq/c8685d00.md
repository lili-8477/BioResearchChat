# Standard QC thresholds (MT<20%, genes>200) work well for tumor scRNA-seq

The analysis applied conventional quality control filters (mitochondrial percentage <20%, genes per cell >200 and <7500) that successfully cleaned the tumor dataset while retaining 9,126 high-quality cells. These standard thresholds proved appropriate for tumor tissue, which can contain stressed or dying cells with elevated mitochondrial content. The QC step was essential before SCimilarity annotation to ensure embeddings weren't distorted by low-quality cells. For tumor samples specifically, monitoring the 20% MT threshold is important as tumor microenvironments often contain more stressed cells than healthy tissues.
