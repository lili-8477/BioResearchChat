# SCimilarity Embeddings Enable Biologically Coherent Clustering Independent of Labels

Using Leiden clustering on SCimilarity embeddings (rather than raw gene expression) produced clusters that showed strong concordance with annotation labels while still revealing biological heterogeneity. The key insight is that SCimilarity embeddings capture cell state similarity in a way that's useful beyond just KNN annotation—they provide a biologically meaningful latent space for downstream clustering and visualization. UMAP parameters of n_neighbors=15 and min_dist=0.1 on these embeddings produced clear cell type separation. This dual validation approach (reference-based annotation + unsupervised clustering) provides confidence that annotations reflect true biological structure rather than just nearest-neighbor artifacts.
