# SCimilarity Achieves High-Confidence Annotations on Tumor Data Without Extensive QC

The tumor scRNA-seq dataset required minimal quality control filtering (only 48/9174 cells removed, 0.5%) yet achieved 93.87% mean annotation confidence with zero low-confidence cells. This suggests SCimilarity's pretrained embeddings are robust to moderate data quality variations. The key QC thresholds that worked well were: <20% mitochondrial reads, 200-7500 genes per cell, and genes expressed in ≥3 cells. The high confidence across 34 distinct cell types validates that reference-based transfer learning can handle complex tumor microenvironments without aggressive filtering that might remove rare but biologically important cell states.
