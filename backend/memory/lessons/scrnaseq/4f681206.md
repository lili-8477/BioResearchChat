# High annotation confidence (>93%) validates SCimilarity's effectiveness on tumor data

The analysis achieved a mean annotation confidence of 93.62% across 9,126 cells, demonstrating that SCimilarity's pretrained model transfers well to tumor single-cell data. The high confidence scores, combined with 33 distinct cell types identified (including tumor microenvironment components like fibroblasts, endothelial cells, and immune cells), suggest that the reference atlas adequately covers the biological diversity in tumor samples. This validates using SCimilarity for rapid cell type annotation without requiring manual marker gene curation or custom training for tumor datasets.
